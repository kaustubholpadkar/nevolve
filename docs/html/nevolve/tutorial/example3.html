Saving and Loading Populations.:


1. Saving Population:

Step 1: Create the Population.
from nevolve.evolve.population import Population
from nevolve.envs.cart_pole_v0 import CartPoleV0

model = Population(
    cls=CartPoleV0,
    size=100,
    mutation_rate=0.05,
    max_workers=10
)
Step 2: Start Evolution.
model.evolve(
    max_generation=100,
    show_best=True
)
Step 3: Save the Population.
model.save_population(path="trained_population.pkl")
Step 4: Save the Best agent of the Population.
model.save_best_(path="best_population.pkl")

2. Loading Population:

Step 1: Create the Population.
from nevolve.evolve.population import Population
from nevolve.envs.cart_pole_v0 import CartPoleV0

model = Population(
    cls=CartPoleV0,
    size=100,
    mutation_rate=0.05,
    max_workers=10
)
Step 2: Load pre-trained population from file.
model.load_population(path="trained_population.pkl")
Step 3: Save the Population.
model.save_population(path="trained_population.pkl")
Step 4: Load the pre-trained Best agent of the Population from file.
model.load_best_(path="best_population.pkl")
