Applying Neuro-Evolution on Sample Environments:


CartPole Environment:

Step 1: Import Population and CartPoleV0 classes.
from nevolve.evolve.population import Population
from nevolve.envs.cart_pole_v0 import CartPoleV0
Step 2: Create Population instance.
model = Population(
    cls=CartPoleV0,
    size=100,
    mutation_rate=0.05,
    max_workers=10
)
Step 3: Start Evolution.
model.evolve(
    max_generation=100,
    show_best=True
)
